
## To contribute:

 * Fork
 * Add a post under `/_posts` (follow format of other posts)
  * The name is in the format: `YYYY-MM-DD-{SLUG}.md`
  * Before the content of your post:

            ---
            title: Your post's title
            layout: post
            author: YOUR NAME
            authorurl: http://YOUR WEBSITE? (or leave blank)
            ---
		 
 * Send pull request
